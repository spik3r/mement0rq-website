/resources

A bookmark dump for fellow new hunters — the things that taught me the most. Curated, not exhaustive.

learning · the foundation

If I were starting over, I'd go through these in order before submitting a single report.

PortSwigger Web Security Academy academy

Free, structured, hands-on. The single best entry point to web vulns. Do the labs.

↗
OWASP WSTG spec

The vendor-neutral checklist of how to test a web app. Reference-grade.

↗
HackTricks wiki

The "how do I exploit X?" reference. Searchable, opinionated, updated often.

↗
PayloadsAllTheThings repo

Payload & bypass cookbook. Cite it once and you'll have it bookmarked forever.

↗

writeups · how good hunters think

Reading writeups taught me more than any course. Steal patterns, then make them yours.

HackerOne Hacktivity feed

Filter by program / severity to find disclosed reports worth dissecting.

↗
Pentester Land list index

Massive aggregator of public writeups, indexed by vuln class.

↗
Infosec Writeups blog

Newer hunters publish here often. Quality varies; the gems are worth it.

↗
PortSwigger Research research

Where new web attack classes get born. Read the originals.

tools · what i actually run

reconprobingscanningbrowserfuzzing
subfinder recon

Passive subdomain enumeration. The first tool I run, every time.

↗
httpx probe

Fast HTTP probing. Status, title, tech, redirects, screenshots — one pass.

↗
nuclei scan

Template-driven vuln scanner. Community templates do 80% of the boring work.

↗
Burp Suite proxy

The interception proxy. Community edition is enough to start; Pro pays for itself fast.

↗
Caido proxy

Modern alternative to Burp. Snappier UI, great workflow features.

↗
katana crawl

Modern crawler. Headless mode catches JS-rendered endpoints other crawlers miss.

↗
ffuf fuzz

Fast web fuzzer. Directories, params, virtual hosts — the swiss-army knife.

↗
gau archive

Pulls known URLs from Wayback / Common Crawl / OTX. Free historical attack surface.

↗

community · where the good takes are

HackerOne · programs platform

Where most public programs live. Sort by bounty, filter by scope.

↗
Bugcrowd · engagements platform

Worth running side-by-side with H1 — different programs, different scope.

↗
LiveOverflow youtube

Deep, calm, technical. Great for building intuition over time.

↗
NahamSec youtube

Bug bounty workflow + interviews with top hunters. Practical.
